Addendum

  • Java script is used to create the functions for lightout game.
  • Create Board() function is used to create a grid of mentioned shape.
  • toggle square() function is used to place the cursor on particular box to off/on the grid.
  • toggle neighbour() function is used to active the neighbour grid by calling the function in toggle square().
  • check win() funcytion is used to check if user won and print message uing windows alert.
  • reset board() function is used to reset the values in the board after choosing a grid.
  • Randomize solvableboard() function is used to solve grid after certain moves .
  • The size of the grid is set to 5 to display 5*5 size grid.
  • The css file is linked to html file to display the grid styles.
  • Java script file containing these function is linked to html file.
  • The html file contains div tag with id named board as reference to initiate function calls.
  • ul tag is used for the bullet points for addendum.
  • The html file is created and javascript and css file are linked to that file for execution.
